# Beenom

## Description

Fronted app (a mini store for buy pills of bee venom) building with [React](https://github.com/facebook/react) and [Redux Toolkit](https://github.com/reduxjs/redux-toolkit)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run dev

# build
$ npm run build

# preview
$ npm run preview
```
